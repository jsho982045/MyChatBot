# MyChatBot
My attempt at trying to make a chatbot using openai api with imessage GUI

To use:
  - Create a .env file
  - Add OPENAI_API_KEY=" Your OpenAI API Key here " 
  - openai.api_key = os.getenv("OPENAI_API_KEY")
